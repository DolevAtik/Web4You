# 🚀 Web4You - אתר מודרני וממיר גבוה

אתר יוקרתי וממיר גבוה לחברת בניית אתרים ודפי נחיתה. עכשיו בגרסת React עם Vite ו-Tailwind CSS.

## ✨ תכונות

✅ **React + Vite** - ביצועים גבוהים ופיתוח מהיר
✅ **Dark Mode יוקרתי** - עיצוב חדשני עם אנימציות חלקות (Framer Motion)
✅ **טופס אפיון חכם** - הטופס המלא ביותר שתצטרך
✅ **שילוב וואטסאפ** - Floating button וכפתורי CTA
✅ **שליחת מיילים אוטומטית** - קבלת טפסים ישירות לדוא״ל שלך דרך FormSubmit
✅ **סקשן פורטפוליו** - הצגת עבודות עם אנימציות
✅ **Mobile First** - רספונסיבי מלא (נייד + דסקטופ)
✅ **Tailwind CSS** - עיצוב מודרני וקל לעדכון
✅ **אנימציות חלקות** - Scroll animations ו-hover effects

## 🛠 התקנה והפעלה

כדי להריץ את האתר במקומי, עקבו אחר השלבים הבאים:

### 1. התקנת תלויות
ודאו שיש לכם Node.js מותקן, ואז הריצו:
```bash
npm install
```

### 2. הרצה במצב פיתוח
כדי לראות את האתר בדחיפה חיה:
```bash
npm run dev
```
ואז כנסו לכתובת: `http://localhost:5173`

### 3. בנייה לפרודקשן
כדי לייצר את הקבצים להעלאה לאינטרנט:
```bash
npm run build
```

## 🎨 התאמה אישית

### שינוי תוכן
מרבית התוכן נמצא בקובץ `src/data/content.js`. זהו המקום המרוכז ביותר לעדכון טקסטים, יתרונות ופורטפוליו.

### שינוי צבעים ופונטים
הצבעים מוגדרים דרך Tailwind CSS. תוכלו לשנות אותם ב-`tailwind.config.js` או ישירות בקומפוננטות ב-`src/components/`.

### שינוי מספר וואטסאפ
חפשו את `972509237173` בקבצים והחליפו למספר שלכם בפורמט בינלאומי.

## 🗂️ מבנה הפרויקט

```
Web4You/
├── src/
│   ├── components/     ← קומפוננטות React
│   ├── data/           ← תוכן האתר (content.js)
│   ├── hooks/          ← לוגיקה מותאמת
│   ├── App.jsx         ← הקומפוננטה הראשית
│   ├── main.jsx        ← נקודת הכניסה של React
│   └── index.css       ← עיצובים גלובליים
├── public/             ← נכסים סטטיים (תמונות וכו')
├── index.html          ← נקודת הכניסה ל-Vite
├── tailwind.config.js  ← הגדרות Tailwind
└── package.json        ← תלויות ופקודות
```
